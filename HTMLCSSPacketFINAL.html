 _____________________________ (HTML): the _________ of a website

HTML Syntax:

Vocabulary

Tags We’ve Learned

Tags: Container for things on your page. The tag name tells the computer what something is. Ex. a <button> tag tells a computer that anything inside of it is a button.


Self-closing Tags: A tag that doesn’t contain text and does not have a closing tag. Ex. a line break, <br>

Attributes: Define characteristics or properties of your tag. Ex. the height or width of an image

<button>: button
<h1>: heading
<p>: paragraph (text)  
<strong>: makes text stand out
<em>: makes text emphasized
<body>: everything you see on your website goes in here
<title>: names the tab
<a href=“ ”>: link
<img src=“ ” alt=“ ”>: image
<br>: line break
<ul>: a bulleted list
<li>: each item in the list

____________________(CSS): the _____________ of a website

CSS Syntax:

Terminology
Selector: Elements you want to change

  • id: Selects one specific tag
  • ex: #fred or #_____________
  • class: Selects anything that has the same class name
  • ex: .color or .____________
  • tag: Selects all of one type of element
  • ex: body or ______________

Property: Aspects of element you want to change

Value: What you want to change it to
